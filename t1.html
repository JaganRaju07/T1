Booking a Cab

    User should provide their user details for loging the application
    They Should provide their registered mobile number to use the Infocab Application
    User should provide their Pickup point location and Destination Address to book a cab

Booking Details

    USer should provide a verified location for booking the cab
    They should verify the ratings and reviews of the cab driver after booking the cab in Infocab Application
    After booking the cab USer and cab driver will communicate by their given registered mobile number

Cancel Booking

    When the driver does not arrive to the destination to the destination user can cancel the cab
    after booking the cab user can check their ratings and cancel the cab
